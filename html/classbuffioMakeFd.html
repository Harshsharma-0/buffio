Detailed Description

Fd maker for buffio.

buffioMakeFd is a helper wrapper to create socket, pipe,fifo or opening files easily, without dealing with low level stuff.

Supported type of FD in buffioMakeFd

  • Network socket
    • local
    • ipv4
    • ipv6
  • pipe

  • fifo
